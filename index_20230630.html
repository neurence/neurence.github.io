Fine-tuning HuBERT network on speech resynthesis task

We're running a new experiment where the encoder (the HuBERT transformer network) is no longer frozen, but is fine-tune trained along with the decoder.

It's hoped that fine-tuning the encoder to the speech synthesis task will improve the autoencoding pipeline. The outputs are shown in the last column of the tables below.
Note that training using fine-tuning is very slow, and the results are shown at an early epoch (40).
