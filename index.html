An autoencoder using HuBERT and HifiGAN

HuBERT is a similar fully-causal Transformer network to Wav2Vec. It seems to be better at speech audio re-synthesis. Its main differnece from Wav2Vec is in the way it learns its phoneme-like lexicon using K-Means clustering interleaved with the context training rather than Wav2Vec's quantization using Gumbel-softmax.

We're currently running experiments training a decoder of our own design to resynthesize audio using a pretrained large (24-block) HuBERT network as the encoder.
The decoder is based on HiFi-GAN.

The HuBERT model has a slightly larger embedding size than Wav2Vec (1024 as opposed to 768). Both have an initial latent feature encoder.

The results below show attempts at resynthsis from the final state (i.e. block 24) and the zeroth block (i.e the output of the initial feature encoder). Included are clean, noisy and denosied inputs for a male and female speaker from our validation set.

The denoised inputs were generated from our current best-performing denoising model (based on Facebook's Demucs).
